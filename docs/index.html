Utils



This utility provides basic utility functions which are mainly used in day to day coding and not ment perform various utility operations which are handled by other utility lib.

Getting started:

npm i tutility --save

Import the js as required
import { deepCopy, isEqual, ... } 'tutility' for utilities
import Detect 'tutility' for browser detect utility
import Logger 'tutility' for logger
