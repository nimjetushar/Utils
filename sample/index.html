Utility Sample Page

The library (umd) is loaded onto the webpage. Use console for accessing various functions of the library. Usage of umd version may differ. Different modules can be invoked as mentioned :-

  • Utility Module -> utils.'name of function' eg. utils.deepCopy(param1, param2)
  • Logger module -> new logger.default() create class instance for Logger
  • Detect module -> new detect.default() create class instance for Detect
